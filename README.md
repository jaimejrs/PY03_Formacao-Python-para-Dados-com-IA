# Formação Python Avançado para Dados: Fundamentos de Engenharia de IA

Repositório de estudos e projetos da **Formação Python para Dados com IA**, com foco em análise de dados, engenharia de dados, cloud, machine learning e IA generativa.

##  Objetivo da formação

Capacitar você para atuar em projetos de dados ponta a ponta, desde a manipulação e exploração dos dados até a criação de pipelines, modelos preditivos e agentes inteligentes com LLMs.

##  Estrutura do repositório (O que foi feito até agora)

O repositório está organizado por aulas e projetos práticos:

- `aula01` a `aula04`: notebooks e atividades introdutórias.
- `aula05/projeto01`: primeiro projeto aplicado com análise em notebook e script Python.
- `aula05/projeto02`: segundo projeto aplicado com automação e envio de e-mail.
- `data`: bases de dados utilizadas nos exercícios.

##  Trilhas de aprendizado da formação

### 1) Fundamentos de Dados

- **Python para Análise de Dados**  
  Manipulação de dados com bibliotecas essenciais como **NumPy** e **Pandas**.
- **Análise Exploratória de Dados (EDA)**  
  Técnicas para investigar, resumir e gerar insights sobre os dados.

### 2) Infraestrutura e Orquestração

- **Contêineres com Docker**  
  Criação de ambientes isolados, reproduzíveis e portáteis para desenvolvimento.
- **Bancos de Dados Relacionais**  
  Uso de **PostgreSQL** e **MySQL** para dados estruturados.
- **Automação com Apache Airflow**  
  Orquestração de pipelines e workflows de dados com DAGs.

### 3) Engenharia e Cloud

- **Big Data e Integração em Larga Escala**  
  Estratégias para processar grandes volumes de dados com eficiência.
- **Bancos de Dados NoSQL**  
  Aplicações com **MongoDB** para dados semiestruturados e não estruturados.
- **Armazenamento em Nuvem AWS**  
  Arquiteturas com **Amazon S3** (Data Lake) e **Amazon Redshift** (Data Warehouse).

### 4) Ciência de Dados e Machine Learning

- **Modelagem Preditiva**  
  Técnicas de **classificação** e **regressão** para previsão de resultados.
- **Algoritmos Avançados**  
  Construção de modelos com **Scikit-learn** e **XGBoost**.

### 5) IA Generativa e Agentes

- **Domínio de LLMs e Prompt Engineering**  
  Boas práticas para interagir com modelos de linguagem de forma eficiente.
- **Arquitetura RAG (Retrieval-Augmented Generation)**  
  Enriquecimento de respostas com dados externos e conhecimento específico.
- **Criação de Agentes Inteligentes**  
  Desenvolvimento de aplicações interativas com **LangChain** e **Streamlit**.
